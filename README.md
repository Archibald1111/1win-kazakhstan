Ассортимент танымал және аз танымал спорт түрлерін, сондай-ақ киберспорт пәндерін қамтиды. Футбол дәстүрлі түрде ең жақсы – 1200-ден астам ставка опциялары. Қалғаны азырақ өзгермелі түрде жазылады.
